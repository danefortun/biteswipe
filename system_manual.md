## Minimum Hardware Requirements

- Any dual-core processor with SSE3 capability
- 2 GB of RAM
- 10 GB storage
- Network interface card with Ethernet capability
- Operating system from this list:
  - Windows 7 or later
  - macOS 12 Monterey or later
  - 64-bit Linux distributions including:
    - Ubuntu 18.04+
    - Debian 10+
    - openSUSE 15.5+
    - Fedora Linux 39+

## Minimum Software Requirements

- Python 3.9 or newer
- Flask WSGI web application framework and its dependencies     (should automatically be installed with Flask)
  - Werkzeug
  - Jinja
  - MarkupSafe
  - ItsDangerous
  - Click
  - Blinker
- Latest compatible version of web browser of choice for testing
