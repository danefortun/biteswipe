# BiteSwipe

BiteSwipe is a social media website for interest discovery. Use this manual for installation to a web server.

## Minimum Hardware Requirements

- Any dual-core processor with SSE3 capability
- 2 GB of RAM
- 10 GB storage
- Network interface card with Ethernet capability
- Operating system from this list:
  - Windows 7 or later
  - macOS 12 Monterey or later
  - 64-bit Linux distributions including:
    - Ubuntu 18.04+
    - Debian 10+
    - openSUSE 15.5+
    - Fedora Linux 39+
